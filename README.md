# HOW TO INSTALL
Copy these files into the main game directory:
```
d3d8.dll                 <- asi loader
libtcc.dll               <- tiny c compiler
scriptHook.asi           <- the plugin
scripts                  <- scripts & sdk
```
(Only works with RomeTW.exe v1.5)
